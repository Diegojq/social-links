# Mi Linktree Personalizado ![Status badge](https://img.shields.io/badge/status-Completed-green)

⚡️ Este proyecto es una implementación práctica de mis habilidades en Tailwind CSS. Desarrollé este Linktree personalizado como parte de un desafío de [Frontend Mentor](https://www.frontendmentor.io/challenges/social-links-profile-UG32l9m6dQ), que me permitió consolidar y demostrar mis capacidades en diseño y desarrollo frontend.

✨ El propósito de este proyecto es agrupar todos mis enlaces de interés en un solo lugar, facilitando así la presentación de mi portafolio y redes sociales de manera organizada y estéticamente agradable.

🔗 [Ver Demostración](https://social-links-nine.vercel.app/)

![image](https://github.com/Diegojq/social-links/assets/137741139/2159b49b-887f-4a78-ae1c-5e0f2ecbedd3)


## 🛠️ Construido con

- HTML5: para la estructura base de la página.
- Tailwind CSS: para estilizar la página de forma moderna y responsiva.
- Desafío de Frontend Mentor: como marco de trabajo para aplicar y demostrar habilidades en un escenario real.
